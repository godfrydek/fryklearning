Logaritmy

Logaritmy od základů: co znamená logₐ x, převod mezi exponenciálním a logaritmickým tvarem, pravidla pro počítání a graf.

Definicelogₐ x = y ⇔ aʸ = x
Podmínkya > 0, a ≠ 1, x > 0
Součinlogₐ(xy) = logₐx + logₐy
Podíllogₐ(x/y) = logₐx - logₐy
Mocninalogₐ(xⁿ) = n·logₐx
Základlogₐ a = 1 · logₐ 1 = 0

Co je logaritmus

Logaritmus odpovídá na otázku: na jakou mocninu musím umocnit základ a, abych dostal číslo x? Například log₂ 8 = 3, protože 2³ = 8.

Převod tvaru

Nejdůležitější je převod logₐ x = y ⇔ aʸ = x. Když se zasekneš, vždy to přepiš do exponenciálního tvaru.

Podmínky

Základ logaritmu musí být kladný a nesmí být 1. Argument logaritmu musí být kladný. Tedy a > 0, a ≠ 1 a x > 0.

Pravidla

Logaritmus součinu je součet logaritmů, logaritmus podílu je rozdíl a exponent můžeš stáhnout před logaritmus.

Graf

Graf y = logₐ x pro a > 1 roste, pro 0 < a < 1 klesá. Vždy prochází bodem (1, 0) a má svislou asymptotu x = 0.

Co umět v testu

Převádět mezi logaritmickým a exponenciálním tvarem, počítat jednoduché hodnoty, znát podmínky a používat základní pravidla.

Přepis je cheat code

Kdykoliv nevíš, co s logaritmem, přepiš ho do exponenciálního tvaru. log₂ 32 = x znamená 2ˣ = 32. Najednou je příklad mnohem čitelnější.

Logaritmy a grafy

Logaritmus je inverzní k exponenciále. Proto logaritmus roste pomalu a má svislou asymptotu x = 0. Nikdy nejde doleva přes nulu, protože argument musí být kladný.

Pravidla nejsou náhodná

Součin se mění na součet, protože exponenty se při násobení stejných základů sčítají. Logaritmy jen přepisují pravidla exponentů do jiné řeči.
