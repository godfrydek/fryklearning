Kvadratické rovnice, funkce a grafy

Kvadratické rovnice, funkce a grafy: diskriminant, kořeny, parabola, vrchol, minimum/maximum a čtení z grafu.

Kvadratická rovniceax² + bx + c = 0
DiskriminantD = b² - 4ac
Kořenyx = (-b ± √D) / 2a
Kvadratická funkcey = ax² + bx + c
Parabolaa > 0 nahoru, a < 0 dolů

Kvadratická rovnice

Tvar ax² + bx + c = 0. Důležité je, že a nesmí být 0, jinak by nešlo o kvadratickou rovnici.

Diskriminant

D = b² - 4ac. Když D > 0, jsou dva reálné kořeny. Když D = 0, jeden dvojnásobný. Když D < 0, žádný reálný kořen.

Myšlenková mapa tématu

Rychlý vizuální přehled před testem.

Kvadratická funkce

Tvar y = ax² + bx + c. Graf je parabola. Když a > 0, je otevřená nahoru a má minimum. Když a < 0, je otevřená dolů a má maximum.

Graf paraboly

Z grafu poznáš kořeny v průsečících s osou x, průsečík s osou y, vrchol a osu souměrnosti.

Co umět v testu

Použít diskriminant, určit počet řešení, vypočítat kořeny, poznat z grafu minimum/maximum a chápat význam kořenů.

Čtení paraboly

Když je parabola otevřená nahoru, má minimum. Když dolů, má maximum. Kořeny jsou průsečíky s osou x, průsečík s osou y získáš z hodnoty c.

Diskriminant jako rychlá kontrola

D > 0 znamená dva reálné kořeny, D = 0 jeden dvojnásobný, D < 0 žádný reálný kořen. Z grafu to vidíš podle kontaktu s osou x.

Mini tahák před testem

  • Nejdřív přečti pojmy nahoře a diagramy.
  • U každého obrázku si řekni vlastními slovy, co znamená.
  • Pak dej rychlý test na 20 otázek a zjisti slabinu.
  • U doplňovaček piš odpovědi oddělené čárkou.
  • U psaných odpovědí test uznává i rozumné formulace, nemusíš psát definici doslova.

Co z toho musíš umět

Umět poznat základní pojem, vysvětlit ho vlastními slovy, použít ho v příkladu a nenechat se nachytat na typické záměny. Test má v tomto okruhu 50 otázek v bance, včetně doplňovaček a bonus režimu.
